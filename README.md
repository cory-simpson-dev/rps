# Real Plur Life

Blog for rave-related content, featuring guides, stories and opinions from users.

[LIVE SITE](https://realplurlife.herokuapp.com/ "Real Plur Life")

## Core Features

- User accounts
- User pages
- Posting
- Direct messaging
- Commenting
- Post/comment editing
- Search/tag filter

## Small Features

- Ad space
- Media (music/video/image) support in articles
- Transforming logo
